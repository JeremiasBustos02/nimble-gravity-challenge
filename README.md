# Nimble Gravity Challenge

Mini aplicación en React que consume la API de Nimble Gravity para listar posiciones abiertas y enviar una postulación.

## Setup

1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Modificar el archivo `.env` con las siguientes variables:

VITE_API_BASE_URL=https://botfilter-h5ddh6dye8exb7ha.centralus-01.azurewebsites.net
VITE_CANDIDATE_EMAIL=tu@email.com

4. Correr el proyecto: `npm run dev`
